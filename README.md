### Built with
- Semantic HTML5 structural elements (`<main>` and `<section>`)

### What I learned
In this second iteration, I focused entirely on code quality and semantics. I learned how to eliminate unnecessary container `div` blocks to keep the HTML tree flat and clean, replaced high-specificity IDs with reusable classes, and streamlined my CSS stylesheet to remove layout duplication completely.
